## 🔧 Основные исправления и улучшения в Форке:

### 🐛 Исправленные ошибки:

1. `wait_until()`: Убран некорректный вызов `datetime.replace()` — теперь используется простое сложение `timedelta`.

2. Парсинг чисел: Вместо `text[-1]` (последний символ) используется регулярное выражение для извлечения полного числа — работает с многозначными значениями.

3. Обработка исключений: Добавлены `try/except` вокруг операций с браузером для предотвращения падений.

## 🎨 Улучшения читаемости:

1. Структура кода: Разделение на логические блоки с комментариями-разделителями.

2. Класс `GameState`: Замена глобальных переменных на инкапсулированное состояние.

3. Docstrings: Добавлены описания функций.

4. Type hints: Указаны типы параметров и возвращаемых значений.

5. Имена переменных: Более понятные названия (`use_snap` вместо `useSnap`).

## ♻️ Рефакторинг:

1. Функция `get_yes_no_input()`: Устранено дублирование кода ввода.

2. `safe_get_element_text()`: Вынесена общая логика получения чисел из UI.

3. `get_quest_location()`: Вынесена логика определения локации квеста.

4. `allocate_skill_points()`: Упрощена и сделана более понятной логика ротации характеристик.
